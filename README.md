How to install colorFabb filament presets and settings on Windows:

STEP 1: Download the presets for your slicer.

STEP 2: Open file explorer.

STEP 3: Type “%appdata%” in the top search-bar, and hit enter.

Step 4: Open the folder with your slicers name.
Currently only BambuStudio, OrcaSlicer and PrusaSlicer are supported.

Step 5: For BambuStudio and OrcaSlicer open the user folder. 
For PrusaSlicer skip to step 7.

Step 6: Open the folder with a numeric name
If there are multiple numerically named folders, repeat the next steps for all these folders!

Step 7: Open either the filament folder for filament presets, or the process folder for print-settings
On PrusaSlicer the folder for print-settings is called “print”

Step 8: Place the downloaded profiles into the correct folders 

Step 9: The profiles should now be installed correctly, happy printing!
